# Assets

This directory contains all static assets used in the course website and materials.

## Directory Structure

- `images/` - Images, diagrams, and screenshots
- `logos/` - Logos and branding materials
- `icons/` - Icon files
- `documents/` - PDF documents, Word files, and other document formats

## Organization

All assets should be organized by type and purpose. Use descriptive filenames and maintain a consistent naming convention.
