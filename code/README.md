# Code Repository

This directory contains all code examples, scripts, and programming materials for the course.

## Directory Structure

- `python/` - Python scripts and Jupyter notebooks
- `colab/` - Google Colab notebooks
- `scripts/` - Utility scripts and helper functions
- `examples/` - Example code snippets and demonstrations

## Organization

Code is organized by week and topic. Each week's code should be in its respective subdirectory.
