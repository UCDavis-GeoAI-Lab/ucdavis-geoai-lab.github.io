# Google Colab Notebooks

Google Colab notebooks for interactive lab sessions and exercises.

## Organization

```
colab/
  ├── week-01/
  │   ├── lab-session-01/
  │   └── lab-session-02/
  ├── week-02/
  └── ...
```

## Usage

Each Colab notebook should be:
- Well-documented with markdown cells
- Include clear instructions
- Be ready to run without additional setup (when possible)
- Link to required data files or resources
